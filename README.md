# Hour Rectification Standalone — Claude Instruction File

A Claude instruction file that helps identify an unknown BaZi birth hour (Hour Pillar). Attach it to any Claude chat, provide the person's date of birth and known life facts, and Claude will systematically narrow all 12 Hour Pillar candidates down to 1–3 most likely options.

No installation, no apps, no prior astrology knowledge required.

---

## How to use

1. Download `hour-rectification-standalone-skill.md` from this repo
2. Open [Claude](https://claude.ai) and start a new chat
3. Attach the `.md` file
4. Send this prompt:

```
Follow the skill instructions in this file and help me rectify the birth hour for [name], DOB [date of birth], [Male/Female].
```

---

## What Claude will do

- Calculate and calibrate the 3-pillar BaZi chart from the date of birth
- Generate all 12 possible Hour Pillars
- Eliminate structurally incompatible hours
- Cross-reference known life facts (siblings, children, health, career, life arc)
- Ask targeted questions to narrow the candidates further
- Return a shortlist of 1–3 most likely hours with reasoning

---

## What helps get a better result

- Sibling count and genders
- Number of children (and any complications)
- Known illnesses and where in the body
- Whether a rough birth window is known (morning, evening, night)
- Major life events with known years
